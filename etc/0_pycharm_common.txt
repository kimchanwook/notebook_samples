-Find any action /  classes/ files / symbols
    -Ctrl + Shift + A
    -Shift + Shift

-Show main main (only in linux)
    -Ctrl + Alt + Shift + z

#--------------------------------------------------------------------------------------
# Editor edits
#--------------------------------------------------------------------------------------

-Show setting
    -Ctrl + Alt + S

-Collapsing
    -Always use Selection folding. Everything else could be confusing
        -Ctrl + .
    -Last line of the block must be un-commented line

    -Collapse/open current blocks
        -Ctrl + -
        -Ctrl + =

    -Collapse/Open all code cells
        -Ctrl + Shift + -
        -Ctrl + Shift + =

-Comment out
    -Ctrl + /

-Increase/Decrease/reset font sizes
    -Ctrl + Alt + Shift + =
    -Ctrl + Alt + Shift + -
    -Ctrl + Alt + Shift + 0


#--------------------------------------------------------------------------------------
#Suggestions
#--------------------------------------------------------------------------------------

-Show suggestion
    -Ctrl + space

-Shows inaccessible classes and members as well as static fields and methods.
    -Ctrl + space + space
    
-all classes throughout the project, regardless of the dependencies
    -Ctrl + space + space + space

-Fix problem - Context Actions / Intention Actions
    -Alt + Enter

#--------------------------------------------------------------------------------------
# Run code or lines
#--------------------------------------------------------------------------------------

-Run code cell in python console
    -Ctrl + Enter

-Run python console of selected lines only
    -Alt + Shift + E

-Run the whole script in TERMINAL
    -Shift + F10
